Climate Exhibit CO2 Display
========

## Introduction

AngularJS to run the CO2 display in the climate exhibit


## Installation

use bower to install the following

angular

angular-route

angular-sanitize

d3

## Instructions

Prior to display, use fileread.php to make a local copy of clean data for each of the three data feeds
